.. _ResetToggles:

ResetToggles
==============================================================================
Sets all hotkeys that contain Toggle back to their first actions.

**Syntax**::

    <ResetToggles>

**Parameters**

    None.

**Example**::

    <Hotkey F1>
       <SendPC local>
          <ResetToggles>
